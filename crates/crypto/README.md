# Fluentbase Crypto

`fluentbase-crypto` provides cryptographic primitives and curve/hash integrations used by
Fluentbase runtimes and contracts.

The crate exposes host/runtime adapters through `CryptoRuntime` and is built to work in
WASM and `std` environments used by the Fluentbase stack.

This crate is part of the [Fluentbase](https://github.com/fluentlabs-xyz/fluentbase) project.
